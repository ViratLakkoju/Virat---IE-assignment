Information Engineering Assignment




Course Description


Overview of electronics and computer organization; design of algorithms and data structures; computational complexity; the Internet; the World Wide Web; server side scripting; client side scripting; databases; encryption and decryption; machine learning; quantum computation.


Course Outcomes


By the end of the course, we will be able to:

1. Designing the algorithms, analyzing its complexity

2. Writing simple web programming scripts

3. Learning about the Interface with database systems

4. Building and deploying machine learning systems
